# hello-python

A simple calculator library in Python.

## Fuctions

- `add(a, b)` — returns a + b
- `subtract(a, b)` — returns a - b
- `multiply(a, b)` — returns a * b
- `divide(a, b)` — returns a / b (raises ValueError if b is 0)

## Usage

```python
from calculator import add, divide

print(add(2, 3))     # 5
print(divide(10, 2)) # 5.0
```
